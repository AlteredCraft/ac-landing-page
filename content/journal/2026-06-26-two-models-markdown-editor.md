I'm lucky to be able to head to the coast this week — writing this from a balcony overlooking Haystack Rock (I'll spare you the Instagram pic).

This week I kicked off the [Cascadia Builders Club](https://luma.com/cascadia-bc). It should be a lot of fun. We're welcoming people demoing apps, but I'd really love to get hardware hackers out to the events too. Join us if you're in the area.

Otherwise, I'm continuing to refine Tilth. Right now I'm running an experiment: can deepseek-v4-flash as the worker and v4-pro as the evaluator build a markdown editor for macOS, orchestrated with Tilth? So far I have a working text editor, and I'm adding markdown rendering as I type this. The bill so far: about $0.41 USD.
