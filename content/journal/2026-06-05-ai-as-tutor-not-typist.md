I've been learning Rust in public, and the interesting part isn't the language. It's how I'm using AI to do it.

The easy path with an AI coding tool is to let it write the code and move on. That gets you a working program and teaches you almost nothing. So for this project I flipped the default: a custom output style that pushes the model to explain its reasoning first, and to hand me less. AI as tutor, not typist.

The tracker itself is a small terminal UI built with ratatui. It's deliberately modest. The point is the practice around it: read the explanation, type the code myself, ask why when something doesn't fit. Slower, and that's the feature.

I'll keep posting what works and what doesn't as I go.
